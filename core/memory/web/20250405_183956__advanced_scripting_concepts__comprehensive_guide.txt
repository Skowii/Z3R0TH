A. Modular programming is a programming paradigm that involves breaking down a program into smaller, independent modules or functions. Each module performs a specific task and can be reused in other programs, making the code more maintainable and scalable. What is object-oriented programming? A. Object-oriented programming (OOP) is a programming paradigm that involves creating objects that contain data and functions. OOP allows for code reuse, encapsulation, and abstraction, making it easier to manage complex codebases. What are libraries and frameworks? A. Libraries and frameworks are pre-written code modules that developers can use to save time and effort in their projects. Libraries provide specific functions and features that can be integrated into a project, while frameworks provide a structure for building entire applications.
