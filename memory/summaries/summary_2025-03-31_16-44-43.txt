Sylius is an open-source eCommerce solution built with decoupled components and a powerful API. The Locale Component manages locales in PHP applications. Documentation for the component can be found on docs.sylius.com, while contributing information can be found on contributing.sylius.com. To follow Sylius' development, follow the official blog, Twitter account, or Facebook page. Bug tracking is managed through GitHub issues, and the project operates under the MIT License.
