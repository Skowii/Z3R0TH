The Sylius E-Commerce PHP component is a solution for managing reviews in an Open Source eCommerce platform. It features powerful API and high-quality code. Documentation is available at docs.sylius.com, and contributing information can be found at https://docs.sylius.com/en/latest/contributing/index.html. For updates and latest features, follow Sylius' official blog, Twitter, or Facebook page. Bug tracking is done on GitHub issues, and the MIT License can be found here. The component was created by Mateusz Zalewski and Grzegorz Sadowski, based on Daniel Richter's work, with a list of contributors available at https://github.com/Sylius/Review/contributors.
