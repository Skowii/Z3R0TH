Maximize your productivity in a bash terminal by using these keyboard shortcuts:

- Ctrl+C: Terminate command
- Ctrl+Z: Suspend command
- Ctrl+A: Move to start of line
- Ctrl+E: Move to end of line
- Ctrl+F: Move forward one character
- Ctrl+B: Move backward one character
- Ctrl+U: Delete from cursor to the start of the line
- Ctrl+K: Delete from cursor to the end of the line
- Ctrl+W: Delete from cursor to the start of word
- Ctrl+D: Delete the character under cursor
- Ctrl+H: Delete the character before cursor
- Ctrl+T: Swap the character under cursor with previous one
- Ctrl+L: Clear screen
