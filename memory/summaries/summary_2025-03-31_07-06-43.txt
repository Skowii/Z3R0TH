El patrón de diseño Spatial Partition ayuda a localizar eficazmente los objetos almacenándolos en una estructura de datos organizada por sus posiciones. Este patrón evita comprobar la posición de cada objeto con todas las demás, reduciendo complejidad y mejorando rendimiento. Estructuras de datos comunes para este patrón incluyen Grid, Quadtree, Árbol K-d, BSP y Jerarquía de volúmenes límite. Se aplica cuando se necesita mantener un registro de posiciones de objetos actualizados y cuando es aceptable cambiar memoria por velocidad.
