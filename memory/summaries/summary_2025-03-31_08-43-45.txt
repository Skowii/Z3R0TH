This folder contains example code related to a RealPython tutorial on using Python's Mock Object Library, focusing on unittest.mock in test file "test_holidays.py." To use this, create a Python virtual environment and install the requirements. Run the tests using `python test_holidays.py` to ensure all pass. Practice by experimenting with code examples, breaking them, and learning how to effectively utilize unittest.mock for testing in Python. The project is licensed under MIT license.
