SyliusAddressingBundle is a bundle that simplifies address management in ecommerce applications. It provides models and architecture for addresses, countries, and provinces with easy customization options. The bundle comes with a set of forms for common actions and supports the flexible Zones system for grouping countries and provinces. Sylius is an Open Source eCommerce solution built from decoupled components with powerful API and the highest quality code. The SyliusAddressingBundle can be found on github, where its license can also be viewed.
