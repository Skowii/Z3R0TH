Sylius Taxation Bundle is a customizable taxation system designed for Symfony2 e-commerce applications. It is part of Sylius, an Open Source eCommerce solution that features a powerful API and high-quality code. Documentation can be found on docs.sylius.com, while contributing information is available on sylius.com/contributing. Follow updates via the official blog, Twitter, and Facebook pages. The bundle was created by Paweł Jędrzejewski and is licensed under MIT.
