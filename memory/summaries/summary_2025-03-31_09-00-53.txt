Bytearray can be created with different methods: an empty bytearray with no values (bytearray()), a zero-filled bytearray of a specified length (bytearray(5)), a bytearray from an iterable of integers (e.g., bytearray([65, 66, 67])), or from a bytes-like object/string using specific encodings and error handling. Examples include bytearray(b"Espa\xc3\xb1ol"), bytearray("Español", "utf-8"), and bytearray("Español", "ascii", errors="ignore"). Bytearray can also be created from hexadecimal digits using the fromhex() method.
