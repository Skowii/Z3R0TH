Le Balking est un modèle de conception qui empêche un objet d'exécuter un certain code s'il se trouve dans un état incomplet ou inapproprié. L'objectif est de permettre une action sur un objet uniquement lorsque celui-ci se trouve dans un état particulier. Par exemple, une machine à laver ne lance le lavage que si elle est à l'état ENABLED. Dans ce modèle, une méthode appelante peut invoquer une méthode sur un objet uniquement si cet objet se trouve dans un état approprié. Ce modèle permet d'assurer une sécurité et une cohérence de fonctionnement entre les différents éléments du programme.
