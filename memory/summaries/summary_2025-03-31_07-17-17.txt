API Gateway es una capa adicional entre el cliente y los microservicios que ayuda a agregar llamadas a estos microservicios. Al hacer una única llamada a la API Gateway, esta entidad se encarga de realizar las llamadas necesarias a los microservicios individuales, lo que evita problemas de actualización y mejora el rendimiento al no necesitar varias solicitudes de red por microservicio. La API Gateway también puede incluir funciones adicionales como análisis de datos, caché y soporte para seguridad y cumplimiento normativo.
