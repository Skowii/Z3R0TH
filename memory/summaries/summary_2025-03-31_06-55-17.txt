El patrón de creación Propósito se utiliza para crear una jerarquía de objetos y nuevos objetos utilizando objetos existentes como padres. Esto permite la personalización dinámica sin alterar la estructura del objeto. En el ejemplo proporcionado, un software moderno en la creación y personalización de teléfonos inteligentes se utiliza para crear configuraciones únicas a medida que los componentes se actualizan. Este patrón se resume en una interfaz Prototype que define operaciones sobre objetos, y su implementación Character realiza la lógica necesaria para trabajar con propiedades dinámicas.
