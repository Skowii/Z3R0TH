Determine if a numeric array is sorted in ascending or descending order using an efficient algorithm in JavaScript. The function checks the direction of the array and returns -1 for descending, 1 for ascending, or 0 if not sorted. Example: `isSorted([0, 1, 2, 2]) // 1; isSorted([4, 3, 2]) // -1; isSorted([4, 3, 5]) // 0; isSorted([4]) // 0`. The function can be modified to work with non-numeric arrays by using `localeCompare` for strings or a custom comparator function.
