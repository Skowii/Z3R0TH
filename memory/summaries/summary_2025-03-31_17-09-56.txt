SyliusAttributeBundle is a bundle for attributes management in Symfony2 applications, which works with the Sylius eCommerce solution. This Open Source tool enables powerful API and high-quality coding. The project's documentation can be found at docs.sylius.com. Contributions are encouraged through GitHub issues, while following official blog, Twitter, or Facebook pages will keep you updated on the latest features.
