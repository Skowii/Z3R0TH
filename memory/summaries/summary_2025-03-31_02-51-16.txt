Inverting key-value pairs of a JavaScript object is possible when values are unique. The following method can be used to invert unique key-value pairs: `Object.fromEntries(Object.entries(obj).map(entry => entry.reverse()))`. If values contain duplicates, the following approach can be used: `Object.entries(obj).reduce((acc, [key, val]) => { acc[val] = acc[val] || []; acc[val].push(key); return acc; }, {})`.
