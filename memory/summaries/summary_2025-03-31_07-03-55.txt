El patrón Prototype es un patrón de diseño de creación en el desarrollo de software que se utiliza para especificar los tipos de objetos a crear a partir de una instancia prototipo y para crear nuevos objetos copiando este prototipo. Se puede utilizar en lugar de crear un objeto desde cero y configurarlo, permitiendo crear una copia de un objeto existente y modificarla según las necesidades. En Java, el patrón Prototype se implementa a través de una interfaz con un método para clonar objetos y una jerarquía de clases que extienden de la interfaz Prototype. Las clases pueden ser especializadas para diferentes tipos de criaturas, como mages, warlords y beasts, y se pueden crear diversos tipos de objetos a partir de prototipos utilizando una fábrica de heroes.
