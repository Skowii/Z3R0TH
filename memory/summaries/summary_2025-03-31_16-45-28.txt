Sylius is an Open Source eCommerce solution built with decoupled components and a powerful API, featuring the highest quality code. Documentation can be found on docs.sylius.com. To contribute, follow instructions on contributing.sylius.com, and stay updated by following official blog, Twitter, or Facebook. Bug tracking is done through GitHub issues, and the MIT License is available at github.com/Sylius/Sylius/blob/master/LICENSE. The component was originally created by Paweł Jędrzejewski and has multiple contributors.
