JavaScript provides two methods for encoding characters to URL-safe strings: `encodeURI()` and `encodeURIComponent()`. The `encodeURIComponent()` function encodes all characters in a string except A-Z, a-z, 0-9, - _, . ! ~ * ', ( ). Use it if the string is only part of a URL. `encodeURI()`, on the other hand, encodes everything except A-Z, a-z, 0-9; , / ? : @ & = + $ - _ . ! ~ *, ( ), #. Use it if the string is a full URL.
