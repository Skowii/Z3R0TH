SyliusOrderBundle is a Sales order bundle for Symfony2 ecommerce applications. It provides sensible and flexible models, default forms, controllers, entities, mappings, and more to help build e-commerce solutions. Sylius is an Open Source eCommerce solution built from decoupled components with powerful API and the highest quality code. The documentation is available on docs.sylius.com, and contributions are welcome by following the contributing guidelines. Follow Sylius' development through their official blog, Twitter, Facebook page, or GitHub issues for bug tracking. The bundle was originally created by Paweł Jędrzejewski and has multiple contributors.
