SyliusPaymentBundle is a payments component integration for Symfony2 eCommerce applications. It is part of Sylius, an Open Source eCommerce solution built with decoupled components and high-quality code. Documentation can be found on docs.sylius.com, and contributions are welcome through GitHub issues. The bundle was originally created by Paweł Jędrzejewski, with a list of contributors available on the project's GitHub page.
