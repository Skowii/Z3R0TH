Sylius product catalog supports various product attributes, options, and variants, along with an archetyping system for efficient creation of similar products. Sylius is an Open Source eCommerce solution built from decoupled components and offers powerful API and high-quality code. Documentation is available on docs.sylius.com, and contributors can follow the contributing guidelines on sylius.com/en/latest/contributing/index.html. Follow Sylius' development through their official blog, Twitter, Facebook page, or report bugs via GitHub issues.
