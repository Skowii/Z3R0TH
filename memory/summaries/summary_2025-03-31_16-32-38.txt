Sylius is an E-Commerce platform for PHP based on the Symfony framework. It's version 1.10.7, has a MIT license and it's maintained by authors like Paweł Jędrzejewski and contributions from the community. The software requires php 7.4 or 8.0 to be installed. It also depends on several other libraries including API Platform Core, Behat Transliterator, Doctrine Collections, Dbal, among others. The platform includes features such as a fixture bundle, grid, mailer, resource, and a theme bundle.
