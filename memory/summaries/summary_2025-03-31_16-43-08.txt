Sylius is an open-source eCommerce solution with decoupled components and powerful API, built with the highest quality code. It integrates various components into a PHP eCommerce framework. Documentation is available on docs.sylius.com, and users can contribute by following guidelines on contributing.sylius.com. Follow official blog, Twitter, and Facebook channels to stay updated on the latest features. Bug tracking is done via GitHub issues, and the MIT License is available for reference. The original creator of the component is Paweł Jędrzejewski, with a list of contributors found on GitHub.
