Create a horizontally or vertically scrollable image gallery using CSS and JavaScript. For vertical scroll, create a flex container with `justify-content: center`, slides with `flex-direction: column`, and use `scroll-snap-type: y mandatory` and `overscroll-behavior-y: contain`. Use `Element.scrollTo()` for smooth scrolling. For horizontal scroll, apply similar principles using `flex-direction: row` and `scroll-snap-type: x mandatory`.
