This directory contains documentation for Sylius, an open-source eCommerce platform built on Symfony. The documentation is hosted on docs.sylius.com and maintained by Read the Docs. Follow the official Twitter account to stay updated. Issues are tracked through GitHub issues. To build the documentation locally, install pip, Sphinx, and requirements using their respective commands. View the generated HTML files in the "build" directory after running the sphinx-build command. The project has many contributors listed on GitHub.
