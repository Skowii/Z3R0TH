Альтернативные названия: Abstract Factory
Путь проектирования: Creational
Язык: ru
Теги: - Gang of Four

Цель: Предоставление интерфейса для создания семейств взаимосвязанных или взаимозависимых объектов без указания их конкретных классов.

Объяснение: Пример из реального мира - королевство с замком, королем и армией; Фабрика фабрик, предоставляющая интерфейс для создания семейств взаимосвязанных или взаимозаменяемых объектов без указания их конкретных классов.

Программный пример: Основываясь на примере королевства, создаем интерфейсы и реализации для объектов в королевстве и абстракцию фабрики королевства с реализацией.
