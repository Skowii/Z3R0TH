This folder contains completed JupyterLab notebooks and files from the Real Python tutorial on using JupyterLab. The files are optional but can be used to further learning. To use them, copy them into a dedicated Samples/ folder and access them through JupyterLab's file browser. To set up, create a virtual environment, install dependencies, and start the JupyterLab server. Load any file by double-clicking on it from the file browser. The information sources for various files are listed in the text.
