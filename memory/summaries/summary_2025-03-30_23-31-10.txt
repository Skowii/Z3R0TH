Initialize a 2D array in JavaScript using Array.from() and Array.prototype.map(). For a specific value: `initialize2DArray(width, height, val = null)`; for an n-dimensional array: `initializeNDArray(val, ...args)`; for a map function: `initializeMapped2DArray(width, height, mapFn = () => null)`; and for skewed 2D arrays: `initializeSkewed2DArray(height, widthFn = () => w, mapFn = () => null)`.
