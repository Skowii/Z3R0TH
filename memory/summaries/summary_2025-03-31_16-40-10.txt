Sylius is an open-source eCommerce solution built with decoupled components and a powerful API, offering high-quality code. Documentation is available at docs.sylius.com, and contributions can be made by following the guidelines on this page. Follow official channels like the blog, Twitter, and Facebook for updates. Bug tracking is done through GitHub issues, while the project uses MIT License. It was originally created by Paweł Jędrzejewski, with a list of contributors available at Sylius' repository.
