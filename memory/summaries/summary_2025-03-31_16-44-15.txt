Sylius is an open-source eCommerce solution built with decoupled components and a powerful API, offering high-quality code. The Currency Component is a currency converting engine for PHP objects. Documentation and contributing information can be found at docs.sylius.com and by following official channels like the blog, Twitter, Facebook, and GitHub issues. The license is available here, and the component was originally created by Paweł Jędrzejewski with contributions from others.
