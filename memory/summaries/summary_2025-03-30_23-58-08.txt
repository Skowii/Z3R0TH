Calculate the date of yesterday, today or tomorrow in JavaScript using the `Date` object. To find the current date, use the `Date` constructor: `const today = () => new Date();`. For yesterday's date, decrement the current date by one: `const yesterday = () => { let d = new Date(); d.setDate(d.getDate() - 1); return d; };`. For tomorrow's date, increment the current date by one: `const tomorrow = () => { let d = new Date(); d.setDate(d.getDate() + 1); return d; };`.
