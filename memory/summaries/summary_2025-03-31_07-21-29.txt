El patrón Objeto Tipo permite la creación flexibilidad de nuevas clases mediante una única clase, donde cada instancia representa un tipo diferente de objeto. Se utiliza en entornos donde se requiere modificar o crear nuevos tipos sin necesidad de modificar el código y recompilar el juego. Un ejemplo común es en videojuegos, donde se necesita agregar o modificar características de diferentes personajes o objetos sin alterar el flujo principal del programa. Se utiliza una clase para los objetos de tipo y otra para los objetos tipados. Los atributos específicos de cada tipo se almacenan en un archivo JSON, y se instancian los tipos necesarios a medida que se requieran.
