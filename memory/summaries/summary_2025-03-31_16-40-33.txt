Sylius e-commerce Shipping component is a comprehensive solution for managing shipments, shipping methods, and calculating shipping costs for modern PHP applications. It offers models for Shipment, ShipmentUnit, ShippingMethod, etc., supports complex ShippingRules systems, and has a flexible Calculators engine to implement various formulas. Sylius is an open-source eCommerce solution built from decoupled components with powerful API and high-quality code. Documentation is available on docs.sylius.com, and contributions can be made through GitHub issues.
