Debugging Node.js using Chrome Developer Tools is possible from v6.3.0. Here's a brief guide:

1. Install Node.js version 6.3.0 or higher if not already installed on your machine.
2. Run node with the '--inspect-brk' flag (e.g., `node --inspect-brk index.js`).
3. Open 'about:inspect' in a new Chrome tab.
4. Click 'Open dedicated DevTools for Node' to launch a window connected to your Node.js instance.
5. Utilize the Developer Tools to debug your Node.js application.
