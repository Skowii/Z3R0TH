The Sylius UI Bundle is a generic user interface (UI) bundle for the Sylius eCommerce components. It is designed for creating shops, ecommerce stores, and webshops. The bundle requires PHP 7.4 or 8.0, as well as several dependencies including Doctrine Collections and Symfony components. This bundle's authors include Paweł Jędrzejewski, the Sylius project, and community contributors. The tests for this bundle utilize PHPUnit, Matthiasnoback's dependency injection test, and phpspec.
