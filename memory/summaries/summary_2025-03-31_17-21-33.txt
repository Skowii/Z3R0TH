The Create Talking Avatar Video block is an AI tool using D-ID service to generate a video of a digital avatar speaking a given script with customizable voice, presenter, and visual settings. It inputs API Key, Script Input, Provider, Voice ID, Presenter ID, Driver ID, Result Format, Crop Type, Subtitles, SSML, Max Polling Attempts, and Polling Interval to create a video with the desired specifications. The output provides a Video URL or an Error message in case of failure.
