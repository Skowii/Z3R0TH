Sylius is an open-source eCommerce solution built from decoupled components with a powerful API and high-quality code. The Taxation component helps handle various items, zones, and tax rates, providing basic models and calculators for tax processing in PHP projects. Documentation is available at docs.sylius.com, and contributions can be made through GitHub issues. The MIT License can be found here, and the original creator is Paweł Jędrzejewski, with a list of contributors available on GitHub.
