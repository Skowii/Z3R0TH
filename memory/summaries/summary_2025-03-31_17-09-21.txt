SyliusReviewBundle is an Open Source eCommerce solution built from decoupled components with powerful API and the highest quality code. It offers documentation on docs.sylius.com and can be contributed to through following the instructions on sylius.com/contributing. Follow Sylius' development updates on the official blog, Twitter, and Facebook. Bug tracking is done via GitHub issues, and authors include Mateusz Zalewski, Grzegorz Sadowski, and Daniel Richter. The license can be found at github.com/Sylius/Sylius/blob/master/LICENSE.
