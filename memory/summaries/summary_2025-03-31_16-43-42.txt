Sylius is an open-source eCommerce solution built from decoupled components with a powerful API and high-quality code. Documentation is available on docs.sylius.com, and contributing information can be found on the contributing page. Follow updates through their official blog, Twitter, or Facebook page, and report bugs using GitHub issues. The project is licensed under the MIT License and was originally created by Paweł Jędrzejewski, with a list of contributors available.
