El Single Table Inheritance Pattern es una técnica que representa jerarquías de clases en una sola tabla, donde todas las subclases comparten los mismos campos de datos. Esta técnica emula la herencia orientada a objetos en una base de datos relacional. Para distinguir entre las diferentes entidades y subclases en una misma tabla, se utiliza una columna discriminadora llamada DTYPE que contiene el nombre de la entidad. La anotación @DiscriminatorColumn se usa para personalizar la columna discriminadora, mientras que @DiscriminatorValue se aplica en las subclases para indicar su valor en la columna product_type.
