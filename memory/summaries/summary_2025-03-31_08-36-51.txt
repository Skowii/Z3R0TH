The project welcomes contributions, but all participants must adhere to the Contributor Code of Conduct. New links should be suggested with source code resources at least 90 days old and having more than 50 stars. Non-source code resources need at least one link to positive discussions indicating strong community backing.

Pull requests should follow specific guidelines, such as no duplicate links and proper formatting. If introducing new categories or improving the README, feel free to disregard the given structure. Maintainers may request changes to existing PRs before merging them.
