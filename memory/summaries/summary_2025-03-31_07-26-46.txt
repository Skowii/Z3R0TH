Feature Toggle (también conocido como Feature Flag) es una técnica utilizada en el desarrollo de software para controlar y gestionar el despliegue de características o funcionalidades específicas sin cambiar el código. Se utiliza como un interruptor de encendido/apagado de funciones basándose en criterios como el estado o las propiedades de otros valores del programa. Esto es similar a las pruebas A/B, donde las funciones se despliegan en función de propiedades como la ubicación o el dispositivo. La implementación de este patrón puede aumentar la complejidad del código y es importante eliminar codigo redundante si se utiliza para eliminar gradualmente un sistema o característica. En entornos ágiles, el Feature Toggle se utiliza en producción para activar la función bajo demanda, para algunos o todos los usuarios.
