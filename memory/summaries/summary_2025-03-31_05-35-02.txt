L'aggregator microservice permet d'appeler plusieurs services pour réaliser une fonctionnalité requise par l'application. Il collecte des éléments de données provenant de divers microservices et renvoie un agrégat pour le traitement. Dans un exemple de programme, un `Aggregator` microservice est créé avec des clients `ProductInformationClient` et `ProductInventoryClient` pour appeler les microservices respectifs et renvoyer l'information combinée. Le modèle de microservices agrégateur peut être utilisé lorsque besoin d'une API unifiée pour différents microservices, quel que soit l'appareil client.
