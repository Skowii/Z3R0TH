SyliusTaxonomyBundle is a flexible categorization system for Symfony2 eCommerce applications based on Spree Taxonomies. It's part of the Sylius project, an open-source eCommerce solution with powerful API and high-quality code. Documentation is available at docs.sylius.com, and contributors can follow the contributing guidelines. For updates and features, follow official blog, Twitter, and Facebook pages or GitHub issues for bug tracking. The bundle was originally created by Paweł Jędrzejewski and has multiple contributors.
