This project, TheAlgorithms/C++, is a collection of algorithms implemented in C++. It has been developed with CMake version 3.22 and uses C++17 standard. There are additional warnings and errors, and an option to use OpenMP for multithreading. A Doxygen-based documentation system has been utilized to generate comments, and CPack is used for package creation. The project contains a variety of subdirectories focusing on different algorithms and topics including math, sorting, searching, bit manipulation, machine learning, physics, etc.
