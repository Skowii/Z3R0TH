Git submodules are a feature that lets you include other repositories within your own repository, useful for libraries or separate code. To add a submodule, use `git submodule add <upstream-path> <local-path>`. After adding a submodule, initialize and update it with `git submodule update --init --recursive`. To update submodules to the latest version, run `git submodule update --recursive --remote`. Renaming a submodule is done with `git mv <old-submodule> <new-submodule>`.
