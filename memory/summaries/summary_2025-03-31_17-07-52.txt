SyliusProductBundle is a product system for Symfony2 applications, supporting simple catalogs and customizable variants, options, and properties. Sylius is an open-source eCommerce solution built with decoupled components, a powerful API, and high-quality code. Documentation can be found on docs.sylius.com, and bug tracking is available via GitHub issues. The bundle was originally created by Paweł Jędrzejewski, with contributions from various authors.
