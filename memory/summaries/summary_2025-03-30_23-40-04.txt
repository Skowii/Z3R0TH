Learn to join an array into a string with appropriate separators using `Intl.ListFormat`. Create formatters using `new Intl.ListFormat(locale, {type, style})`, then use the `format()` method to join the array with locale-sensitive separators (conjunction or disjunction) and style (short, long, or narrow). Example: `formatList(['foo', 'bar', 'baz'], 'en-US', 'conjunction', 'short')` returns 'foo, bar, & baz'.
