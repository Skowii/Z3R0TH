SyliusOrderBundle is a Sales order bundle for Symfony2 ecommerce applications, providing default forms, controllers, entities, mappings and everything needed to build an application. It includes an order and order item total value adjustments system based on the Spree adjustments concept. The documentation can be found at docs.sylius.com and bug tracking is available through GitHub issues.
