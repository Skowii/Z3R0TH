SyliusChannelBundle is a Symfony2 bundle used to handle multiple channels in e-commerce applications. It is part of the Sylius project, an Open Source eCommerce solution with a powerful API and high-quality code. The bundle's documentation can be found on docs.sylius.com, and it uses the MIT License.
