# TAGS: # Contributing to the Docs
- Contributions welcome
- Setting up docs:
  1. Clone repository
  2. Install dependencies
  3. Use mkdocs' live server
- Adding a new page:
  1. Create markdown file in `docs/content`
  2. Update nav section in `mkdocs.yml`
  3. Add content to the new markdown file
  4. Run `mkdocs serve` to see changes
- Checking links: run `mkdocs build` for broken link warnings
- Submitting a Pull Request
