SyliusCurrencyBundle is a Symfony2 integration for Sylius Currency processing component which helps in managing currencies and conversions in an eCommerce solution. The bundle was created by Paweł Jędrzejewski and has active contributors. It uses the MIT license and has good documentation available on docs.sylius.com.
