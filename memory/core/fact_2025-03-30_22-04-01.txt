# LinkUp Full Setup Guide (111) ## Step 1: Update Your System ```bash sudo apt update && sudo apt upgrade -y ``` ## Step 2: Install Required Dependencies ```bash sudo apt install curl git unzip zip tar wget -y ``` ## Step 3: Install Java 17 ```bash sudo apt install openjdk-17-jdk -y java -version ``` ## Step 4: Install Flutter ```bash cd ~ wget https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_3.16.0-stable.tar.xz tar -xf flutter_linux_3.16.0-stable.tar.xz mv flutter ~/linkup/flutter ``` **Add Flutter to PATH:** ```bash echo 'export PATH="$HOME/linkup/flutter/bin:$PATH"' >> ~/.bashrc source ~/.bashrc ``` **Verify Flutter:** ```bash flutter --version ``` ## Step 5: Install Android Studio ```bash cd ~/linkup wget https://redirector.gvt1.com/edgedl/android/studio/ide-zips/2024.1.1.0/android-studio-2024.1.1.0-linux.tar.gz tar -xvzf android-studio-2024.1.1.0-linux.tar.gz mv android-studio ~/linkup/android-studio ``` **Launch Android Studio:** ```bash ~/linkup/android-studio/bin/studio.sh ``` - Complete **setup wizard** and install **SDKs**. ## Step 6: Configure Android SDK & ADB ```bash echo 'export ANDROID_HOME="$HOME/Android/Sdk"' >> ~/.bashrc echo 'export PATH="$ANDROID_HOME/tools:$ANDROID_HOME/tools/bin:$ANDROID_HOME/platform-tools:$PATH"' >> ~/.bashrc source ~/.bashrc ``` **Verify ADB:** ```bash adb version ``` ## Step 7: Install Gradle ```bash sdk install gradle 8.4 gradle -v ``` ## Step 8: Install & Set Up Firebase 1. **Go to Firebase Console**: [Firebase Console](https://console.firebase.google.com/) 2. **Create a new project**: Name it **LinkUp**. 3. **Enable Firestore, Authentication, and Storage**. 4. **Download `google-services.json`** and place it in:    ```    ~/linkup/android/app/google-services.json    ``` ## Step 9: Install Flutter Dependencies ```bash cd ~/linkup flutter create linkup cd linkup ``` Edit `pubspec.yaml`: ```yaml dependencies:   flutter:     sdk: flutter   firebase_core: latest_version   firebase_auth: latest_version   cloud_firestore: latest_version   provider: latest_version ``` Run: ```bash flutter pub get ``` ## Step 10: Set Up Emulator for Testing ```bash flutter doctor ``` **Create an Emulator:** ```bash ~/Android/Sdk/emulator/emulator -avd Pixel_4_API_30 ``` **Run Emulator:** ```bash flutter emulators --launch Pixel_4_API_30 ``` ## Step 11: Set Up Git for Version Control ```bash cd ~/linkup git init git add . git commit -m "Initial commit for LinkUp" ``` To connect with GitHub: ```bash git remote add origin https://github.com/your_username/LinkUp.git git push -u origin main ``` ## Step 12: Run Your App ```bash flutter run ``` For release mode: ```bash flutter run --release ``` ## Step 13: Debugging & Optimization ```bash flutter analyze flutter doctor flutter clean ``` ## Step 14: AI-Powered Admin Dashboard Integration -
