Payment Component is an Open Source eCommerce solution built from decoupled components with powerful API and the highest quality code. It does not provide any gateways but can be integrated with many different services, using Payum by default. Documentation is available on docs.sylius.com and for bug tracking, follow GitHub issues. The component was originally created by Paweł Jędrzejewski and has a list of contributors.
