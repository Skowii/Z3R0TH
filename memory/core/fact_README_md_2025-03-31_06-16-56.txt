The Promotion Component is a powerful marketing tool for e-commerce projects, offering time-based promotions, fixed and percentage discounts, custom actions, rules system, promotion coupons with usage limits. It is part of the Sylius Open Source eCommerce solution built from decoupled components with a powerful API and high-quality code. The documentation can be found on docs.sylius.com, and contributing information can be found here. Follow Sylius' development through their official blog, Twitter, Facebook, or report bugs via GitHub issues. The component was originally created by Paweł Jędrzejewski with contributions from various developers. The project is licensed under MIT License.
