Sylius Core Bundle is a bundle that integrates all other bundles into an eCommerce solution for Symfony2. It's part of Sylius, an open source eCommerce solution built from decoupled components with powerful API and high-quality code. Documentation is available at docs.sylius.com and the software uses the MIT License.
