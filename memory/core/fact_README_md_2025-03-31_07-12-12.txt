SyliusAdminBundle is a bundle that includes the Sylius administration panel for an open-source eCommerce solution with powerful API and high-quality code. It is built from decoupled components, and its documentation can be found on docs.sylius.com. This bundle was originally created by Paweł Jędrzejewski and has contributors listed at github.com/Sylius/Sylius/contributors.
