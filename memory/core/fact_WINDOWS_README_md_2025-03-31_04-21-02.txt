Windows Setup:
1. Install package via cmd or IDE (stable) or git clone and poetry install (development)
2. Set OPENAI_API_KEY environment variable
3. Run gpt-engineer with prompt file in the new project folder
4. Check generated files in workspace
5. Read ToS before usage
