Summary: SyliusInventoryBundle is a flexible inventory management tool for Symfony2 eCommerce applications, heavily inspired by Spree inventory system. It forms part of the open-source eCommerce solution Sylius. Documentation and contributing information can be found on docs.sylius.com, while bug tracking occurs through GitHub issues. The bundle is licensed under MIT License, with the original creator being Paweł Jędrzejewski.
