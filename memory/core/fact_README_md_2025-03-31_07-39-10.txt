Sylius Locale Bundle is a Symfony2 integration of Sylius Locale management component, which is an Open Source eCommerce solution with powerful API and the highest quality code. The documentation for this bundle can be found on docs.sylius.com and the development updates can be followed on Sylius' official blog, Twitter account, and Facebook page. It uses GitHub issues to track bugs and is licensed under MIT License.
