Sylius Documentation is an Open Source eCommerce platform on top of Symfony, available at docs.sylius.com. The documentation uses GitHub issues for reporting any problems or suggestions. You can follow Sylius updates on Twitter and contribute to the project by installing Sphinx to build and view the HTML files in the "build" directory.
